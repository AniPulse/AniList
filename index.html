AniList

A production-ready anime database and tracking app built with vanilla JS, HTML, CSS, and AniList GraphQL API. Features advanced search with filters, personal lists (watching/completed/planning), dark/light themes, responsive design, infinite scroll, and user progress tracking. Deploy easily with GitHub + Vercel.
